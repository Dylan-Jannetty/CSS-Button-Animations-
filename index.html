CSS Button Animations

With the installment of CSS3, we were presented with animation and transformation properties.

  • Shadow
  • Rotate

Shadow Effects

Click on each button to view the code!

  • HTML
  • CSS
              <button class="btn white grow">
              <p>Grow</p>
              </button>
            
    
    .grow {
    background-color: $redtint;
    border: 1px solid $white;
    border-radius: 5px;
    height: 50px;
    margin: 5px 15px 0;
    padding-top: 15px;
    text-align: center;
    transition: .5s;
    width: 150px;
    }
            
    .grow:hover {
      border: 0;
      box-shadow: 1px 1px 20px 1px $black;
      transform: scale(1.15);
      transition: .5s;
    }
  • HTML
  • CSS
            <button class="btn white shrink">
            <p>Shrink</p>
            </button>
          
  
  .shrink {
    background-color: $redtint;
    border: 1px solid $white;
    border-radius: 5px;
    height: 50px;
    margin: 5px 15px 0;
    padding-top: 15px;
    text-align: center;
    transition: .5s;
    width: 150px;
  }
          
  .shrink:hover {
    border: 0;
    box-shadow: 1px 1px 20px 1px $black;
    transform: scale(.8);
    transition: .5s;
  }

Rotate Effects

Click on each button to view the code!

  • HTML
  • CSS
                <button class="btn white rotate-right-Z360">
                <p>Rotate <br> Right </p>
                </button>
              
      
      .rotate-right-Z360 {
        background-color: $redtint;
        border: 1px solid $white;
        border-radius: 5px;
        font-size: 5px;
        height: 50px;
        margin: 15px 15px 0;
        padding-top: 10px;
        text-align: center;
        transition: 1s;
        width: 150px;

        p {
          font-size: 12px;
        }
      }
              
      .rotate-right-360Z:hover {
        transform: rotateZ(360deg);
        transition: 1s;
      }
